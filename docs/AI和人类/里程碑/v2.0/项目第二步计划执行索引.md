## 0. 已钉死口径

### 0.1 范围边界

| 做 | 不做（本步） |
|----|----------------|
| `commonUiComponents` + Formik + Modal + 前端分页 | Host / Engine 写口、真实 `data/` 持久化 |
| 故事包新建/导入 Modal 化、列表分页 | 真实 zip 导入导出管道 |
| 角色/资源/用户 mock 增删改、详情 Formik 编辑 | 登录、权限、玩家运营后台 |
| 编辑器属性面板字段、role 连线、icon dock、悬浮角色轨、添加角色 | 调试器任何改动 |
| 更新 `09` + `studio-v2-quality` 的 commonUi 规则 | 改 `packages/rpg-engine` 业务 |

### 0.2 与第一步的关系

- 第一步（V1-*）已收口：静态壳 + `quality:studio` 门禁。
- 休整计划（REST-*）已收口。
- 本步 **重新定义** V2.0「第二步」为交互深化；第一步索引文末「第二步 = Host」的表述 **以本索引与第二步需求为准**。

### 0.3 推荐刀序

```text
V2-S*  系统横切（Formik、commonUi、Modal、分页、规则、试点）
V2-P*  /packages Modal + 分页
V2-C*  /characters mock CRUD
V2-A*  /assets mock CRUD
V2-U*  /users mock CRUD

```

`group: order`：严格按表中从上到下取 ⬜；每轮最多 `batch_size` 条。
