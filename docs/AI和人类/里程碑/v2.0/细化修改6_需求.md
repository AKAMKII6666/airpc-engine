## 1. 本轮定位

### 1.1 要做什么

```text
0 底栏 toolMode 状态机：idle / placement / connect / select；active icon 高亮
1 视图工具：适配视图 fitView；框选 / 连线 / 平移模式互斥切换
2 放置创建：点底栏「新建 X」→ 光标 crosshair → 点画布空白 → 生成节点 → 自动选中并开属性浮窗
3 五种 CallCard 放置：剧情 / 过场播放 / 自由 / 延迟外呼（对齐 cardKind + entryMode 默认投影）
4 章节结束放置：新增 chapter_end 节点（复用 refine5 章节属性窗）
5 轻量节点：动作节点、注释分组（纯 UI mock，不参与引擎校验）
6 资源 / 包配置 icon：保持 refine4 已有行为，与 toolMode 不冲突
```

### 1.2 本轮不做

| 不做 | 说明 |
|------|------|
| 通话卡 / 包 / 画布写盘 | 不接 `data/storis-packages`、不写 `story.conf.json`、不接 Host save |
| 引擎 schema 或运行时改动 | 动作 / 注释节点为 Studio UI 投影；不新增 `CallCardDefinition` 字段 |
| 从出口拖线到空白弹出创建菜单 | 向导稿 §7.2；另立 refine7 |
| 动作节点完整 Effect 编排 | 动作节点仅标题 + 摘要 mock |
| 注释分组嵌套 / 折叠 / 引擎语义 | 虚线框 + 标题即可 |
| 「运行当前位置」 | 向导稿 §3.4 提及但当前底栏无此 icon |
| 恢复左侧角色轨 / 底栏角色浮窗 | refine4 已 supersede |
| 调试器改版 | 不碰 `DebuggerShell` |

### 1.3 与 refine4 / refine5 的关系

- **依赖** refine4 画布 CallCard / 章节 / 角色锚点、属性浮窗、资源与包配置浮窗。
- **依赖** refine5 章节结束 `nextPackageId` / `nextEntryCardId` 配置。
- **不重复** 角色新建编辑（仍在画布锚点 + `/characters` FormModal）。

## 2. 已钉死口径

### 2.1 底栏 icon 清单与 mock 行为

| icon id | 中文 | mock 行为 |
|---------|------|-----------|
| `add_callcard` | 新建 CallCard | placement → 新增 `cardKind=story` CallCard |
| `playback` | 过场播放卡 | placement → `cardKind=system`，`entryMode=playback`，`interactionMode=playback_only` |
| `free` | 自由通话卡 | placement → `cardKind=free` |
| `delay` | 延迟外呼卡 | placement → `cardKind=schedule`，带默认 `schedule.mode=daily` |
| `chapter_end` | 章节结束 | placement → `kind=chapter_end` 章节节点 |
| `action` | 动作节点 | placement → `type=action` 轻量节点（非 CallCard） |
| `comment` | 注释分组 | placement → `type=commentGroup` 轻量节点 |
| `connect` | 连线 | toolMode=connect：可拖 handle 连线；禁平移抢焦点 |
| `select` | 框选 | toolMode=select：`selectionOnDrag=true` |
| `fit` | 适配视图 | 瞬时调用 `fitView()`，不切持久 toolMode |
| （右侧）`assets` | 资源浮窗 | 保持 refine4 CRUD 浮窗 |
| （右侧）`package` | 包配置 | 保持 refine4 只读浮窗 |

**禁用文案：** 本批完成后，底栏工具 tooltip **不得** 再带「界面预览未开放」。

### 2.2 toolMode 状态机

| 模式 | 含义 | 与底栏关系 |
|------|------|------------|
| `idle` | 默认：平移画布 + 点选节点 | 无 placement / connect / select 高亮 |
| `placement` | 等待画布点击放置 | 对应「新建 X」icon 高亮；Esc 或再点同 icon 取消 |
| `connect` | 连线优先 | `connect` icon 高亮；`panOnDrag=false` 或等价 |
| `select` | 框选优先 | `select` icon 高亮；`selectionOnDrag=true` |

**互斥：** `connect` 与 `select` 互斥；进入 `placement` 时退出 connect/select。  
**fit：** 瞬时动作，执行后回到进入 fit 前的 toolMode（或 idle）。

### 2.3 放置创建流程（对齐向导稿 §7.1）

1. 用户点底栏「新建 X」→ 进入 `placement`，携带 `placementKind`。
2. 画布空白处点击（`onPaneClick`）→ `screenToFlowPosition` 换算坐标。
3. 调用节点工厂生成 **唯一 nodeId / cardId**（禁止用户手填；可用 `createStudioId` 或等价）。
4. 追加节点到 React Flow `nodes`；自动选中；打开属性浮窗。
5. 回到 `idle`（或保持 placement 连放——**第一版放置一次后回 idle**）。

**默认投影（CallCard）：**

| placementKind | cardKind | entryMode | interactionMode | 其它 |
|---------------|----------|-----------|-----------------|------|
| story | `story` | `inbound_user_dial` | `realtime_dialogue` | 空 exits[]、inherit_free toolPolicy |
| playback | `system` | `playback` | `playback_only` | 标题默认「过场播放」 |
| free | `free` | `either` | `realtime_dialogue` | — |
| schedule | `schedule` | `outbound_auto` | `realtime_dialogue` | `schedule: { mode: daily, hour: 10 }` |

**ownerAgentId：** 第一版可空或未设；用户后续通过 role 连线或属性窗补。

### 2.4 轻量节点（action / commentGroup）

**动作节点 `EditorActionNodeData`（建议）：**

- `title: string`
- `summary?: string`
- 视觉：小卡片 + 闪电 icon；**不参与** CallCard 校验 badge。

**注释分组 `EditorCommentGroupNodeData`（建议）：**

- `title: string`
- 视觉：虚线矩形框；可选固定宽高 mock；不参与 story 边引擎语义。

二者 **禁止** 混入 `CallCardDefinition` 或 `EditorCallCardProjection` 类型。

### 2.5 画布 API 扩展

在 `StoryCanvasStageApi` 增加（或等价壳层命令）：

| 方法 | 用途 |
|------|------|
| `setToolMode(mode, placementKind?)` | 底栏切换 |
| `fitView()` | 适配视图 |
| `addNodeAt(kind, position)` | 放置工厂入口 |
| `getToolMode()` | 可选；供 BottomToolBar 高亮 |

实现落点：`bis/pageBis/storyEditor/dock/` + `canvas/` 扩展；**禁止**在 `BottomToolBar` 内直接 mutate RF state。

### 2.6 资源 / 包配置

- 点击资源 / 包 icon **不切换** toolMode；仅打开浮窗（refine4 行为）。
- 浮窗打开时 placement 模式 **取消**（避免误点画布）。
