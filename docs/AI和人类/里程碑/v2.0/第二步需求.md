> **状态：** 已定稿，供批执行与 Review 引用。

## 1. 本步定位

### 1.1 要做什么

在 `apps/studioV2` 已有路由壳与静态页基础上，把各主界面从「能看」推进到 **可操作的高保真原型**：

- 系统级横切：**Formik 统一表单** + **`src/commonUiComponents` 公共 UI** + **Modal** + **前端分页**
- 故事包台：新建/导入 Modal 化、列表分页
- 角色库 / 资源库 / 用户档案：mock 增删改、详情可编辑、字段覆盖完整
- 故事编辑器：属性面板有实质字段、角色归属可连线、底栏 icon dock；~~左侧角色轨悬浮、可添加角色~~ → 见细化修改 4（画布角色节点唯一入口）

### 1.2 本步不做

| 不做 | 说明 |
|------|------|
| Host / Engine 写口闭环 | 不假装已保存到 `data/` |
| 真实导入导出管道 | 导入预检、导出打包可用 mock 结果 |
| 调试器深化 | **整模块不碰** `/debugger` |
| 电话壳 / Realtime | 与第一步边界一致 |
| 引擎改动 | 除非修 V2 门禁误报，本步默认不改 `packages/rpg-engine` |

### 1.3 与第一步收尾说明的关系

第一步执行索引文末曾写「第二步 = 真实保存、Host 对接」。经人类确认（原始需求讨论记录），**V2.0 第二步优先做静态页交互深化**；真实保存 / Host / 调试闭环另开后续里程碑，**不得在本步索引中静默扩入**。

## 3. 系统级横切（必须先于业务模块）

### 3.1 目录：`src/commonUiComponents`

路径：`apps/studioV2/src/commonUiComponents/`。

职责：跨页面复用的 **展示型 UI**（Modal、表单字段、分页等）。  
禁止：barrel `index.ts`；业务编排（进 `bis/`）；类型契约（进 `typeFiles/`）。

**抽离规则（本步写入 09 + studio-v2-quality）：**

- 同一组件被 **≥2 个业务页面** 使用时，必须抽到 `commonUiComponents`
- 单页私有组件仍放 `pageComponents/<route>/com/`

### 3.2 表单：统一 Formik

- 本步新增依赖：`formik`（校验可选 `yup`，第二迭代再加）
- 禁止新业务表单继续裸 `useState` 拼字段（试点迁移 `CreatePackageView` 后为准）
- 字段外壳对标 mithril `formComponentsContainer`：**label + 控件 + 错误 + mode(add/edit/watch)**

建议组件：

```text
commonUiComponents/
  modal/app/AppModal.tsx
  modal/form/FormModal.tsx
  modal/shared/modalTypes.ts
  modal/shared/modalSlot.ts
  form/FormFieldShell.tsx
  form/FormSchemaRenderer.tsx
  form/formTypes.ts
  form/fields/FormTextField.tsx
  form/fields/FormSelectField.tsx
  form/fields/FormCheckboxField.tsx
  form/fields/FormTextAreaField.tsx
  pagination/FrontendPagination.tsx
```

### 3.3 样式与栈

- 仅 MUI + CSS Modules + SCSS（与第一步一致）
- 文案：**全中文**（用户可见处）
- 内部 ID：**系统生成，主流程不手填**（`createStudioId` 等）

### 3.4 横切验收

- [ ] `CreatePackageView`（或等价试点）已走 Formik + `FormSchemaRenderer`
- [ ] `AppModal` / `FormModal` 可在任意页复用
- [ ] `FrontendPagination` 可独立使用
- [ ] `09` 与 `studio-v2-quality.mdc` 已写明 `commonUiComponents` 规则
- [ ] `npm run quality:studio` 全绿

## 5. 数据与状态约定（静态阶段）

- Mock 数据继续放 `src/utils/ajaxProxy/**`；页面通过 `bis/` 或 store 消费，**禁止** mock 散落进 `typeFiles/`
- 增删改：**会话内**可变（Zustand 或模块级 mock 可变数组均可）；不要求刷新后持久
- 不得伪造「已保存到磁盘」的成功 toast 文案
