# 04. 调试器界面需求

路由：`/debugger`。浏览器只展示与发意图；模拟推进、可选文本 LLM 均在 **server**。

相关：[02](./02-运行时协议.md) · [10](./10-用户与存档.md) · [19](../技术设计文档/19-引擎宿主与会话模型.md) · [18](../技术设计文档/18-部署拓扑与BS架构.md) · [50](./50-对话与模型适配.md)

## 1. 进入流程（用户门禁）

```text
进入 /debugger
  → UserGate（全页或 Dialog：列表 / 新增 / 选择 / 删除；无密码）
  → 选定 userId → 加载 PlayerProfile
  → 进入调试台
```

与故事列表共用同一 `UserGate` 组件（见 [03](./03-编辑器界面需求.md)、[19](../技术设计文档/19-引擎宿主与会话模型.md)）。

## 2. 信息架构

```text
┌──────────────┬─────────────────────────────┬──────────────────┐
│ 模拟控制      │     Board / 时间线            │ 当前卡详情        │
│ · 当前用户    │  各角色 pending / active      │ · Composer 快照   │
│ · 起始卡      │  出口命中 / Free 管线         │ · 命中 promptScenes│
│ · 本地时间覆盖│  effect / Profile diff         │ · 工具策略        │
│ · 电话事件    │  log 切片                     │ · 出口+命中原因   │
│ · Outcome    │                               │ · 记忆摘要        │
│ · 时钟快进    │                               │                  │
├──────────────┴─────────────────────────────┴──────────────────┤
│ 底部：WET / Facts&knowledge / Lore / 校验警告 / 错误码提示        │
└──────────────────────────────────────────────────────────────┘
```

## 3. 基础流程

```text
选定 User → 加载 Profile（可重置）
  → 选起始卡 → simulate_start（读已保存 Content，冻结 frozenCard）
  → 模拟电话事件 / 文本聊 / 手动 beats·flags
  → Story：ExitSelector → Effects
    Free：FreeCallPostPipeline（MemoryCommit；有候选则 ExitSelector）
  → save Profile；查看 diff 与 logs
  → 可选快进触发 schedule（触发点 = clock + minMs）
```

对话：v1 **Manual**；可接 **server 文本 LLM** 聊天 UI。Outcome：默认 Manual；高级区可「模型建议 Outcome」再确认（见 [19 §13](../技术设计文档/19-引擎宿主与会话模型.md)）。

**本地时间 / 场景组装：**

- `beginCall` 上报浏览器本地时间（或「固定本地时间」覆盖）→ `ComposeScene`  
- Composer 快照须显示：`callDirection`、本地时段 bucket、**matchedLayerIds**、最终 opening  
- 详见 [50](./50-对话与模型适配.md)

**过程话术模拟：** 按钮触发 `silence_timeout` / `call_duration_threshold` / `pre_hangup_hint`（引擎不计时）。

## 4. 与编辑器严格分离

- 调试器**不提供**改卡 / 改 package 的编辑 UI。
- 本通使用 `CallSession.frozenCard`；画布未保存草稿不影响本通。
- 发现问题：结束调试 → 回 `/stories/[id]` 修改并保存 → 再进调试。

## 5. 必须显示

- Profile.Board diff；命中出口原因（static / dynamic）或 Free 管线步骤
- effect plan：`ok | skipped | failed` + plan.status
- RuntimeExitCandidate（可手动登记模拟 FC）
- world.facts / knowledge / memory 摘要；Lore 状态
- Composer 快照（受显示等级约束）；**命中的 promptScenes layer**
- **日志时间线**（Host ring + 可打开 jsonl 切片）
- 当前 `composeScene.localTime`（可覆盖）

## 5.1 调试显示等级（隐私）

| 模式 | privateBrief |
|------|----------------|
| **author** | 可见 |
| **playtest** | 默认隐藏；可临时揭示 |
| **导出日志** | **不落** private 原文 |

## 6. 电话事件与 Outcome

接通结束、未接、占线、提前挂断、播放完成/跳过、重播、留言已听、回拨触发。预填 flags 后可改 beats，再结束通话。

## 7. 非目标

- 调试器内改卡或热重载本通 frozenCard
- v1 强制 Realtime / 自动 Outcome
- 旧话机进程连接、完整时间旅行（P1）
