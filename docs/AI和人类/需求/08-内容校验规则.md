# 08. 内容校验规则

相关：[01](./01-CallCard数据模型.md) · [13](./13-工具与能力注册.md) · [14](./14-资源与媒体资产.md) · [19](../技术设计文档/19-引擎宿主与会话模型.md) · [03](./03-编辑器界面需求.md)

`validatePackage(packageId)` / `validateWorkspace()` 产出 `ValidationReport`：`{ errors[], warnings[] }`，每项含 `ruleId`、路径、文案。

**级别：** `error` 阻断导出与调试开跑（可配置）；`warning` 可导出，底栏提示。

## 1. 包 / 磁盘

| ruleId | 级别 | 说明 |
|--------|------|------|
| `CONF_CARD_FILE_MISSING` | error | conf `cards[]` 有 cardId，无 `cards/<id>.s-card.json` |
| `CARD_FILE_ORPHAN` | warning | 磁盘有 `.s-card.json` 未登记进 conf |
| `ENTRY_CARD_UNKNOWN` | error | `entryCardId` 不在索引内 |
| `REFERENCED_AGENT_UNKNOWN` | error | 包内派生引用的 agentId（owner / effects / attach 等）不在角色库 |
| `PARTICIPANT_UNKNOWN` | warning | 遗留 `participants` 含不存在的 agentId（字段将忽略）；空数组或不写不报错 |
| `FREE_PACKAGE_SENTINEL` | error | 试图把 `__free__` 当真实故事包目录创建或 load |
| `SCHEMA_UNSUPPORTED` | error | schemaVersion 不兼容（见 [17](../技术设计文档/17-版本迁移与兼容策略.md)） |
| `LAYOUT_IGNORED` | — | layout 旁车不参与运行校验（可存在） |

**路径 B（角色全局化）：** `participants` 可选、非白名单。`validatePackage` 对 `collectReferencedAgentIds` 派生集合做角色库存在性与 free/social 校验；`ownerAgentId` 不在遗留 `participants` 但在角色库 → **不得** error。

## 2. CallCard / 出口

| ruleId | 级别 | 说明 |
|--------|------|------|
| `PROMPT_SCENE_PATCH_HARD` | error | `promptScenes.patch` 含 objective/forbidden |
| `EXIT_EMPTY_STORY` | warning | story 卡完全无 exits；有任意一条出口即不再警告（不强制 failure/recovery 类别）。`cardKind=voicemail` 允许空出口，本规则不检 |
| `EFFECT_UNKNOWN` | error | 未知 effect 名 |
| `EFFECT_ID_DUP` | error | 包内 effect.id 重复 |
| `PLAYBACK_NO_ASSET` | error | playback_only 无 playbackClipId（**不含** `cardKind=voicemail`：留言靠物化，允许无 clip） |

## 2.1 语音留言（voicemail）

真源：[语音留言改造需求](../里程碑/v2.0/语音留言改造需求.md) §3.1–§3.3；实现见 `validation/package/voicemail.ts`。

| ruleId | 级别 | 说明 |
|--------|------|------|
| `VOICEMAIL_INTERACTION_MODE` | error | `cardKind=voicemail` 但 `interactionMode !== playback_only` |
| `VOICEMAIL_ENTRY_MODE` | error | `cardKind=voicemail` 但 `entryMode !== mailbox_open` |
| `VOICEMAIL_TOOL_POLICY` | error | `cardKind=voicemail` 但 toolPolicy 非 deny_all 等价 |
| `VOICEMAIL_ENTRY_KIND` | error | 非 voicemail 卡使用 `entryMode=mailbox_open` |
| `VOICEMAIL_CREATE_EFFECT` | error | 包内仍出现废弃 Effect `create_voicemail`（须改 `attach_call_card` / `schedule_call_card` 指向 voicemail 卡）；对 cardRaw 扫描，schema 亦拒识该 effect 名 |

## 3. 工具

| ruleId | 级别 | 说明 |
|--------|------|------|
| `TOOL_UNKNOWN` | error | allowedToolIds 含未注册 id |
| `TOOL_KIND_MISMATCH` | error | 工具不允许该 cardKind |
| `TOOL_PLAYBACK` | error | playback_only 仍声明工具 |
| `TOOL_DIRECT_EFFECT` | error | 试图标成通话中直接 effect |
| `TOOL_INTRODUCE_GUARD` | warning | 引荐类工具目标无 `canIntroduce` |

## 3.1 Schedule / 外呼意图

落盘分工见 [01 §8.1](./01-CallCard数据模型.md)、[19 §3.4](../技术设计文档/19-引擎宿主与会话模型.md)。

| ruleId | 级别 | 说明 |
|--------|------|------|
| `SCHEDULE_ONCE_CARD_REQUIRED` | error | `schedule_call_card` 缺 `agentId`+`packageId`+`cardId` |
| `SCHEDULE_RECURRING_CARD_REQUIRED` | error | `schedule_recurring_call` 缺 `scheduleCardId` 且缺 `cardId`+`packageId` |
| `SCHEDULE_RECURRING_IN_STORY` | error | StoryCard 上挂 `schedule_recurring_call`（周期外呼归 ScheduleCard 宿主） |
| `SCHEDULE_CARD_KIND` | error | recurring 目标不在 `characters/schedule-cards/`（或非法 `__free__` fallback）；**含**故事包内 `cardKind=schedule` 剧情节点冒充日常卡 |

## 4. 资产 / 角色

| ruleId | 级别 | 说明 |
|--------|------|------|
| `ASSET_UNKNOWN` | error | 引用不存在的 assetId |
| `ASSET_URI_MISSING` | error | uri 指向缺失文件 |
| `ASSET_KIND_MISMATCH` | warning | playback 引用了非音频类 |
| `ASSET_PACKAGE_INLINE` | warning | 包内出现完整 assets[] 内联（应迁全局库） |
| `SOCIAL_TARGET_UNKNOWN` | error | social.target 不在角色库 |
| `NARRATIVE_DIALABLE` | error | `isNarrativeOnly` 但 `dialable: true` |
| `NARRATIVE_NEEDS_NO_FREE` | warning | narrative-only 仍配置了 freeCardId |
| `FREE_CARD_MISSING` | error | 非 narrative-only 可通话角色缺 freeCardId 或文件 |
| `FREE_CARD_KIND` | error | free 卡 `cardKind !== "free"` |
| `CHARACTER_NOT_DIALABLE` | — | 运行时错误码，非包校验 |
