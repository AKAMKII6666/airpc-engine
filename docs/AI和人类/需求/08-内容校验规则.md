# 08. 内容校验规则

相关：[01](./01-CallCard数据模型.md) · [13](./13-工具与能力注册.md) · [14](./14-资源与媒体资产.md) · [19](../技术设计文档/19-引擎宿主与会话模型.md) · [03](./03-编辑器界面需求.md)

`validatePackage(packageId)` / `validateWorkspace()` 产出 `ValidationReport`：`{ errors[], warnings[] }`，每项含 `ruleId`、路径、文案。

**级别：** `error` 阻断导出与调试开跑（可配置）；`warning` 可导出，底栏提示。

## 1. 包 / 磁盘

| ruleId | 级别 | 说明 |
|--------|------|------|
| `CONF_CARD_FILE_MISSING` | error | conf `cards[]` 有 cardId，无 `cards/<id>.s-card.json` |
| `CARD_FILE_ORPHAN` | warning | 磁盘有 `.s-card.json` 未登记进 conf |
| `ENTRY_CARD_UNKNOWN` | error | `entryCardId` 不在索引内 |
| `PARTICIPANT_UNKNOWN` | error | `participants` 含不存在的 agentId |
| `FREE_PACKAGE_SENTINEL` | error | 试图把 `__free__` 当真实故事包目录创建或 load |
| `SCHEMA_UNSUPPORTED` | error | schemaVersion 不兼容（见 [17](../技术设计文档/17-版本迁移与兼容策略.md)） |
| `LAYOUT_IGNORED` | — | layout 旁车不参与运行校验（可存在） |

## 2. CallCard / 出口

| ruleId | 级别 | 说明 |
|--------|------|------|
| `PROMPT_SCENE_PATCH_HARD` | error | `promptScenes.patch` 含 objective/forbidden |
| `EXIT_EMPTY_STORY` | warning | story 卡无 exits（创作建议至少成功+failure） |
| `EXIT_NO_FAILURE` | warning | story 卡缺 failure 类出口 |
| `EXIT_NO_RECOVERY` | warning | story 卡缺 recovery 建议出口 |
| `EFFECT_UNKNOWN` | error | 未知 effect 名 |
| `EFFECT_ID_DUP` | error | 包内 effect.id 重复 |
| `PLAYBACK_NO_ASSET` | error | playback_only 无 playbackClipId |

## 3. 工具

| ruleId | 级别 | 说明 |
|--------|------|------|
| `TOOL_UNKNOWN` | error | allowedToolIds 含未注册 id |
| `TOOL_KIND_MISMATCH` | error | 工具不允许该 cardKind |
| `TOOL_PLAYBACK` | error | playback_only 仍声明工具 |
| `TOOL_DIRECT_EFFECT` | error | 试图标成通话中直接 effect |
| `TOOL_INTRODUCE_GUARD` | warning | 引荐类工具目标无 `canIntroduce` |

## 4. 资产 / 角色

| ruleId | 级别 | 说明 |
|--------|------|------|
| `ASSET_UNKNOWN` | error | 引用不存在的 assetId |
| `ASSET_URI_MISSING` | error | uri 指向缺失文件 |
| `ASSET_KIND_MISMATCH` | warning | playback 引用了非音频类 |
| `SOCIAL_TARGET_UNKNOWN` | error | social.target 不在角色库 |
| `NARRATIVE_DIALABLE` | error | `isNarrativeOnly` 但 `dialable: true` |
| `NARRATIVE_NEEDS_NO_FREE` | warning | narrative-only 仍配置了 freeCardId |
| `FREE_CARD_MISSING` | error | 非 narrative-only 可通话角色缺 freeCardId 或文件 |
| `FREE_CARD_KIND` | error | free 卡 `cardKind !== "free"` |
| `CHARACTER_NOT_DIALABLE` | — | 运行时错误码，非包校验 |

## 5. 验收

- [x] error 阻断导出；点击问题可定位卡/文件（导出 API `hasBlockingErrors`；定位 UX 可增强）
- [x] golden 包 validate 无 error
- [x] `__free__` 哨兵不可当包路径通过校验
