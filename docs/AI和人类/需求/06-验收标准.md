# 06. 验收标准

## 1. 引擎（`packages/rpg-engine`）

- [ ] User / PlayerProfile 创建、序列化；无多存档槽
- [ ] 剧情 / Board / facts / `redialSlot` 在薄 Profile（JSON）；**记忆在 SQLite**（经 MemoryPort），不在 `profile.save.json` 大数组
- [ ] CharacterDef + social + `effectiveDialable`（unlocked 可覆盖）+ `set_character_unlocked`
- [ ] ActiveStoryLock 决策表（见 19）；Resolver 读锁
- [ ] CallCard schema / validatePackage；layout 旁车忽略
- [ ] ToolRegistry：七支业务 + `search_memory`/`get_memory_by_id`；挂机三支不进；Free `inherit_free`
- [ ] RuntimeExitCandidate；工具不直接 Effect
- [ ] CallSession schema；frozenCard；Free `packageId=__free__`；同 user 通话互斥
- [ ] Resolver：pending > Free；priority；schedule = clock+minMs
- [ ] ExitSelector：静态 ∪ 动态；同 priority 静态优先
- [ ] Free → FreeCallPostPipeline（MemoryCommit；有候选再 Exit）；禁隐式 transcript 扫描
- [ ] Free + `refer_to_expert` candidate → 挂机后 unlock/schedule 落盘
- [ ] Effect + EffectSink 时序；无 Relay
- [ ] MemoryCommit 契约（12）：episodic 通末总结 + 生活 vignette；写 SQLite；**必带时间戳**；失败不阻断已执行 Effect
- [ ] `projectForCall` 有条数/字符预算；禁止 beginCall 全表灌 prompt
- [ ] `search_memory` / `get_memory_by_id`：`session_local`、硬 limit、FTS 查库（非向量）；Hit≤200 / 全文≤500；不扫 JSON 文件
- [ ] Commit 后可增量 `rollupIfNeeded`；不在 beginCall 热路径全量重算
- [ ] Effect plan：失败继续、critical 中止；ledger 可测
- [ ] 事件序 + `data/logs/*.jsonl`
- [ ] patch_memory / fact / knowledge 跨角色仅出口
- [ ] schemaVersion 拒载；错误码表（19）；[08](./08-内容校验规则.md) 总表
- [ ] Composer：`ComposeScene`/`RenderedPrompt`/`DialogueSessionSpec`；本地时间；patch 不得盖 objective/forbidden
- [ ] 端口可替换；零 React
